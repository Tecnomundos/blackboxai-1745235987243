# CosmoClick - Guia Espiritual e Astrológico

## Project Overview
**CosmoClick** is an interactive web application designed to guide users through spiritual and astrological insights. By entering their name and birth date, users receive personalized spiritual guidance based on numerology, astrology, and ancestral wisdom. The application also features sections on various spiritual topics, providing an enriching experience for users interested in exploring their spiritual journey.

## Installation
To run the **CosmoClick** application locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://your-repo-url.git
   cd cosmo-click
   ```

2. **Open `index.html` in your web browser:**
   Simply double-click the `index.html` file or open it with your preferred browser.

## Usage
1. Enter your full name and birth date in the provided form.
2. Click the "Revelar Meu Destino Sagrado" button to submit the form.
3. View your personalized spiritual insights in the results section below the form.
4. Explore additional information about ancestral wisdom and guidance for various life areas.

## Features
- **Personalized Spiritual Guidance:** Users can calculate their spiritual insights based on their name and birth date.
- **Multi-language Support:** The application offers language options in Portuguese, English, German, and French.
- **Rich Content Areas:** Includes sections like "Sabedoria Ancestral" and "Guia para Sua Jornada" that provide essential knowledge and tips.
- **Responsive Design:** The application is built using Tailwind CSS, ensuring a visually appealing and responsive user interface.

## Dependencies
The project uses the following libraries and frameworks:
- [Tailwind CSS](https://tailwindcss.com/) for styling.
- [Font Awesome](https://fontawesome.com/) for iconography.
- Google Fonts for typography.
